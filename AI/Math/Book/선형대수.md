# 선형대수

### 1-1. 벡터

- 벡터
  - 데이터 여러 개를 한 줄에 담아낼 수 있게 만든 것(in 수학)
  - 여러 개의 데이터를 하나의 열에 담아 둔 것(in 프로그래밍)
- 벡터 표기 방법
  - 하나의 문자로 나타내는 방법
  - 구체적인 성분을 나열하는 방법



<br/>

### 1-2. 덧셈과 뺄셈 그리고 스칼라배

- 덧셈
  - 서로 대응하는 성분끼리 더한다.

- 차원: 벡터 성분의 수

- 서로 다른 차원의 벡터는 더 할 수 없다.

- 스칼라: 벡터와 대비되는 개념으로 크기만 있고 방향이 없는 상수나 변수 같은 1차원의 값



<br/>

### 1-3. 유향선분

- 유향선분 = 방향과 거리를 나타내는 화살표

- 벡터를 화살표로 표기하면 벡터에 대한 개념을 이해하는게 더 쉬워진다.



<br/>

### 1-4. 내적

- 내적

  - 벡터에서 서로 대응하는 성분끼리 곱한 다음, 그것들을 모두 더한 것
  - 벡터와 벡터의 내적은 스칼라가 된다.
  - 서로 다른 차원의 벡터끼리는 계산 할 수 없다.

- 기호

  - <**a**, **b**>

- 기하학적 특징으로 정의

  - 벡터 a 와 b가 이루는 각이 θ일 때, <**a**, **b**>는 다음과 같이 정의할 수 있다.
    $$
    \lVert a \rVert \lVert b \rVert cos\theta
    $$
    각 θ는 두 개의 벡터 a와 b의 시작점을 일치시켰을 때 생기는 벡터와 벡터 사이의 각



<br/>

### 1-5. 직교 조건

- 내적이 0인 벡터는 서로 직교하고 있다.

- 두 개의 벡터가 직교한다 = 두 벡터가 이루는 각이 90º = 내적 <**a**, **b**>가 0이라는 것
- 즉, 두 벡터가 직교하는지 알아보려면 내적이 0인지 확인하면 된다.
- 정의
  - 벡터 a, b 가 서로 직교할때, 다음 식이 성립한다.
    - <**a**, **b**> = 0



<br/>

### 1-6. 법선벡터

- 하나의 점과 법선벡터가 주어지면 하나의 평면을 정할 수 있다.
- 접평면
  - 
- 접점
  - 접평면과 접하는 구의 점
- 법선벡터
  - 3차원 곡면에 대한 전체 접선(접평면)과 직교하는 벡터



<br/>

### 1-7. 벡터의 노름

- L1 노름은 벡터 성분의 절대값을 모두 더하면 구할 수 있다.
- L2 노름은 벡터의 유클리드 거리로 구할 수 있다.
- 노름(norm)
  
- 벡터는 방향과 이동 거리가 중요한데 이때의 이동거리
  
- L1 노름

  - ex) **a** = (4, 3)

  - L1 = 4 + 3 = 7

  - 정의

    - a에 대한 L1 노름 = 
      $$
      \sum_{i=1}^n |a_i|
      $$

    

    <br/>

    - 즉, L1을 계산할 때는 벡터의 각 성분들의 절대값들의 합을 구하면 된다.



- L2 노름

  - 시작점에서 목적지까지 직선으로 움직이는 방법

  - 유클리드거리와 같다.

  - 정의

    - a에 대한 L2 노름 = 
      $$
      \sqrt {a_1^2 + a_2^2 + \cdots + a_n^2}
      $$

  - 내적으로 표현

    - 벡터 a에 대한 L2 노름 = 
      $$
      \lVert a \rVert_2 = \sqrt { \left\langle a, a \right\rangle}
      $$

  

<br/>

- L1 노름과 L2 노름은 Linear regression 모델의 정규화(regularization)항에서 사용한다.



<br/>

### 1-8. 코사인 유사도

- 코사인 유사도는 -1 이상 1 이하의 값을 가진다.

- cosθ

- cos(**a**, **b**)
  $$
  = \frac{	\left\langle a, b \right\rangle}{	\lVert a \rVert 	\lVert b \rVert}
  $$
  <br/>

- 코사인 유사도가 -1일 때, 두 벡터가 서로 반대 방향으로 평행
- 코사인 유사도가 0일 때는 직교 상태
- 코사인 유사도가 1일 때는 서로 같은 방향으로 평행

<br/>

- 코사인 유사도가 높다는 말은 벡터가 더 비슷하다는 의미 



<br/>

### 1-9. 행렬의 덧셈과 뺄셈

- 행렬의 덧셈과 뺄셈은 서로 대응하는 각 성분을 있는 그대로 더하고 빼면 된다.
- 행렬은 같은 차원의 벡터를 모아서 사각형 모양으로 나열한 것이다.
- 행(가로), 열(세로)
- 성분
- 행렬 표기
  - A, B, ...



<br/>

### 1-10. 행렬의 곱셈

- 행렬의 곱셈은 곧 벡터의 내적을 확장한 것

- m x n 행렬과 n x 1 행렬(n차원 열벡터)을 곱한것

  = m개의 n차원 행벡터와 

- 일반적으로는 교환법칙(AB=BA)이 성립하지 않으나 예외는 있다.

- 영행렬(zero matrix)

  - 행렬의 성분 전체가 0인 행렬

- 단위행렬(identity matrix)

  - 왼쪽 위에서 오른쪽 아래 방향으로 대각선상의 모든 성분이 1이고, 그 밖의 성분은 0으로 채워진 정방행렬

  - ex)
    $$
    \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}
    $$
    

<br/>

### 1-11. 역행렬

- 행렬에는 나눗셈이 없는 대신 역행렬이 있다.
- 역행렬은 정방행렬로만 정의된다.
- 예를들어 3으로 나누는 것은 3의 역수(1/3)를 곱하는것과 같다.
- 이런 역수의 개념을 행렬에 적용한 것이 역행렬이다.

 <br/>

- 행렬 A, 역행렬 A^-1, 단위행렬 E

- 정의

  - $$
    A A^{-1} = A^{-1}A = E
    $$

    

- 행렬 A와 A^-1의 곱이 단위행렬로 나오려면 이 행렬들도 정방행렬이어야 한다.

- 정방행렬

  - 행의 개수와 열의 개수가 같은 행렬

- 정방행렬이라 하더라고 행렬 A의 행렬식이 0인 경우에는 역행렬이 존재하지 않는다.

- 행렬식(determinant)

  - det A or |A|

  - $$
    \left| A \right| = det A = ad - bc
    $$

    

- 2 x 2 행렬의 역행렬을 구하는 공식
  $$
  A = \begin{pmatrix} a & b \\ c & d \end{pmatrix} \text{에 대하여}
  $$

  $$
  A^{-1} = \frac{1}{ad-bc} \begin{pmatrix} d & -b \\ -c & a \end{pmatrix}
  $$

 ### 1-12. 선형 변환

- 수학적으로 벡터에 행렬을 곱해 또 다른 벡터를 만드는 함수

- 표준기저(standard basis) `e`
  - x축이나 y축, 그리고 z축처럼 일종의 좌표계를 정할 수 있는 벡터의 집합
- 선형 변환 또는 1차변환
  - 벡터 공간에서 벡터를 변환하는 것

### 1-13. 고윳값과 고유벡터

- 2 x 2 행렬은 반드시 두 쌍의 고윳값과 고유벡터를 가진다.

- 정방행렬 A가 있고 다음의 식을 만족하는 열벡터 x(단, x ≠ 0)가 존재할 때, 

  λ를 행렬 A의 고윳값(eigenvalue)이라 하고 

  x를 고유벡터(eigenvector)라고 말한다.
  $$
  Ax = λEx
  $$
  (이때, E는 단위행렬)

- 고유방정식(eigenvalue equation)

  - det(A - λE) = 0
  - λ의 방정식을 행렬 A의 고유방정식이라고 말한다.
