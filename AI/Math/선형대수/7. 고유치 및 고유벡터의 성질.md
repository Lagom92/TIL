# 7. 고유치 및 고유벡터의 성질

<br/>

### 고유치 및 고유벡터의 성질

​	1) A 행렬의 고유치의 합은 대각원소들의 합(trace)이다.

​	2) A행렬의 고유치의 곱은 행렬식 값과 같다.



<br/>

- 고유값(eigenvalue)들의 합 = trace



<br/>

- det(λI - A) = 0의 모든 근의 합 

  = 고유방정식의 근의 합

  = 고유치의 합

  = trace



<br/>

- det(A-λI) = 0의 모든 λ들의 곱

  = 고유치의 곱

  = 행렬식의 값





<br/>

### 고유치 및 고유벡터의 성질

​	1) A^T와 A의 고유치는 같다.

​	2) A의 고유치가 λ이면

​		A^n의 고유치는 λ^m 이다.

​		=> A^-1의 고유치는 λ^-1이다.

​			αA의 고유치는 αλ이다.

​			A + αI의 고유치는 λ + α이다.



- 1번 일때는 고유벡터가 같지 않다.

- 2번 일때는 고유벡터가 같다.
